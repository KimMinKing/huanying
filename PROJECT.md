# 라이플(Lifful) — 프로젝트 문서

> 비즈니스/운영 관점에서 프로젝트를 이해하기 위한 문서.
> 개발자용 기술 가이드는 `CLAUDE.md`, 입력 데이터는 `DATA.md` 참고.

## 서비스 한 줄 소개

한국 생활 서비스(인터넷, 휴대폰, 렌탈, 이사, 청소, 보험/카드) 비교·신청 플랫폼.
특히 **중국인/외국인 정착 패키지**와 **AI 맞춤 추천**으로 차별화.

## 타겟 고객

### 1차 타겟
- 한국에 거주하는 일반 소비자 (20~40대)
- 통신/렌탈/이사 견적을 한 곳에서 비교하고 싶은 사람
- 영업 방문 없이 폼으로 신청 끝내고 싶은 사람

### 2차 타겟 (차별화 핵심)
- 한국에 정착하는 중국인/외국인
- 휴대폰 개통, 은행 계좌, 인터넷 설치, 집 구하기를 한 번에 해결하고 싶은 사람
- 중국어(WeChat) 상담이 가능한 채널을 원하는 사람

## 차별점

1. **비교 투명성** — A/B/C사 견적을 동일 기준으로 한눈에. 숨은 비용(위약금, 설치비 등)까지 명시.
2. **중국어 상담** — 카카오톡 외에 WeChat 채널 운영. 중국인 고객 접근성 우위.
3. **AI 맞춤 추천** — 체류기간, 비자, 지역, 예산만 입력하면 30초 안에 맞춤 조합 제안.
4. **정착 원스톱 패키지** — 입국 전 상담부터 6개월 사후관리까지 전담 매니저 1:1 동행.

## 로드맵 (대략적)

### Phase 1 — 현재 (MVP)
- 프론트엔드 랜딩 + AI 데모
- 목업 관리자 페이지
- DB 스키마 설계

### Phase 2 — 백엔드 + 자동화
- Express/Fastify + Prisma 백엔드
- 실제 상담 신청 → DB 저장 → 관리자 페이지 연동
- 알림 (문자/카카오톡 알림톡)
- 파일 업로드 (외국인등록증/여권)

### Phase 3 — AI 서버
- LLM 기반 맞춤 추천/견적 서버 분리
- 상담 매니저용 AI 어시스턴트 (요약, 다음 액션 제안)
- 다국어 자동 번역 채팅

### Phase 4 — 파트너 통합
- 통신사/렌탈사 API 직접 연동 (실시간 요금 조회)
- 자동 청구/정산 시스템
- 파트너사 전용 포털

### Phase 5 — 확장
- 다국어 확장 (영어, 베트남어, 우즈베크어)
- 모바일 앱 (React Native)
- B2B (기업 외국인 직원 정착 서비스)
